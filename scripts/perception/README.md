# Perception Namespace

This namespace is prepared for the multimodal track from `План ИИ.md`.

- `proto/` — gRPC contracts
- `python_service/` — BEATs+CLAP service scaffold (currently placeholder inference)
- `csharp/` — Godot .NET client scaffold
- `perception_router.gd` — routing helper to `GameServer.send_perception_context_to_kim(...)`

Current MVP remains text-first and fully functional without this namespace.
